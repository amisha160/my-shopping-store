# 🛍️ My Shopping Store

*Shopping Website — Project Report*


## Purpose

My Shopping Store is a multi-page static shopping website built to demonstrate core front-end web development skills, including HTML5 structure, CSS styling, and client-side JavaScript. The site simulates a real-world e-commerce platform with consistent design, interactive features, and clean navigation across four distinct pages.


### Home (index.html)

- Hero banner section with gradient background and store branding
- Category listing with emoji icons for Clothing, Electronics, and Accessories
- JavaScript-powered offer button that reveals a 20% discount message
- Navbar with links to all pages

### Products (products.html)

- Product list displaying T-Shirt (₹500), Headphones (₹1500), and Watch (₹2000)
- CSS gradient image boxes with emoji icons for each product
- Clean card layout with product names and prices

### Order (order.html)

- Order form with four fields: Name, Product, Quantity, and Address
- Form submission via POST method to local server
- Simple and clean layout for easy ordering

### Contact (contact.html)

- Contact details: Address (Bangalore, India), Email, and Phone Number
- JavaScript-powered button that displays a thank-you message on click
- Gradient contact image box for visual appeal

---

## Tech Stack

- **HTML5** — semantic markup across all four pages
- **CSS3** — gradients, box shadows, hover effects, and card layout
- **Vanilla JavaScript** — offer reveal (Home) and thank-you message (Contact)

---

## How to Run

1. Clone or download this repository
2. Open `index.html` in any web browser
3. No installation or server required — fully static site
